VAST CHALLENGE 2026 Mini-Challenge 1

Investigation of Merger Information Leak

Did TenantThread’s AI agents deliberately leak the merger — or did the compliance system simply break down under pressure?

On 5 June 2046, confidential merger information between TenantThread and CivicLoom Realty Partners surfaced online at 5:00 PM — exactly one hour before the official embargo lift. We use network analysis, text mining, and interactive visual analytics across 912 internal communications to reconstruct what really happened.

A project by BUI Tien Thanh, HU Yue & WU Jiayan · ISSS608 Visual Analytics & Applications

What we set out to do

The investigation is organised around four analytical objectives, each surfaced as an interactive lens inside the Shiny application:

  1. Reconstruct the event sequence during the breach window (4:00–6:00 PM, 5 June) using network analysis and timeline visualisation.
  2. Analyse behavioural patterns by comparing breach-day behaviour against a baseline to detect anomalies consistent with deliberate action or system strain.
  3. Identify key actors and information flow through centrality analysis to find critical coordinators and bottlenecks.
  4. Apply a competing-hypotheses framework to systematically weigh evidence for a deliberate leak versus a system breakdown.
